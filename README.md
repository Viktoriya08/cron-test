## Начало работы

Установка npm зависимостей

```
npm install
```

Запуск в режиме разработчика

```
npm run dev
```

Собрать проект

```
npm run build
```
Собрать проект с форматированием картинок в WebP

```
npm run buildWithWebP

```

Собрать svg sprite

```
npm run sprite
```

Перезаписать шрифты (Если появились новые)

```
npm run fonts
```
